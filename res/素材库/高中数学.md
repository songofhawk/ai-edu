### **详细课程设计**
#### **模块一：函数与复利（2-3课时）**
**课题1：指数函数与复利计算**

+ **数学知识**：指数函数 $f(x) = a^x$，特别是 $f(x) = (1+r)^x$
+ **核心公式**：$F = P(1+r)^n$
+ **理财应用**：
    - **银行定期存款：10万元，年利率3%，10年后本利和**
    - **基金定投：每年投入1万元，年收益率8%，20年后总资产**
    - **通货膨胀：购买力随时间指数衰减**
+ **教学方法**：
    - **对比不同利率的增长速度**
    - **绘制指数曲线，直观展示复利的威力**
    - **实际计算案例，让学生感受时间的价值**

**课题2：对数函数与投资时间**

+ **数学知识**：对数函数 $\log_a x$，特别是自然对数 $\ln x$
+ **核心公式**：$n = \frac{\ln(F/P)}{\ln(1+r)}$
+ **理财应用**：
    - **投资翻倍需要多少年？**
    - **达到目标资产需要多少年？**
    - **不同利率下的时间对比**
+ **教学方法**：
    - **通过对数反解时间**
    - **对比不同利率的时间成本**
    - **强调"时间就是金钱"的概念**

---

#### **模块二：数列与定期投资（3-4课时）**
**课题1：等差数列与定期定额投资**

+ **数学知识**：等差数列 $a_n = a_1 + (n-1)d$，求和 $S_n = \frac{n(a_1+a_n)}{2}$
+ **理财应用**：
    - **每月定投1000元，12个月后总投入**
    - **投资收益线性增长的情况**
    - **储蓄计划的制定**
+ **教学方法**：
    - **列表展示逐月投入和累计总额**
    - **对比定额投资和不定额投资**
    - **计算不同投资周期的总投入**

**课题2：等比数列与复利投资**

+ **数学知识**：等比数列 $a_n = a_1 \cdot r^{n-1}$，求和 $S_n = \frac{a_1(1-r^n)}{1-r}$
+ **理财应用**：
    - **年收益率固定的投资增长**
    - **每年增加投入的复利增长**
    - **投资组合的长期增长**
+ **教学方法**：
    - **对比等差和等比数列的增长速度**
    - **展示复利的指数增长特性**
    - **强调长期投资的优势**

**课题3：数列求和与贷款还款**

+ **数学知识**：等比数列求和的应用，特别是 $\sum_{i=0}^{n-1} (1+r)^i = \frac{(1+r)^n-1}{r}$
+ **核心公式**：等额本息月付款 $M = \frac{P \cdot r(1+r)^n}{(1+r)^n-1}$
+ **理财应用**：
    - **房贷100万，年利率4.9%，30年等额本息还款**
    - **汽车贷款的还款计划**
    - **贷款总利息计算**
    - **提前还款的决策**
+ **教学方法**：
    - **推导等额本息公式**
    - **制作完整的还款计划表**
    - **对比不同还款方式（等额本息 vs 等额本金）**
    - **计算总利息和实际利率**

---

#### **模块三：不等式与线性规划（2-3课时）**
**课题1：线性规划与投资组合优化**

+ **数学知识**：线性不等式、可行域、目标函数、最优解
+ **问题建模**：
    - **决策变量：$x$（股票投资额），$y$（债券投资额）**
    - **目标函数：最大化收益 $Z = 0.10x + 0.04y$**
    - **约束条件：**
        * **$x + y \leq 100000$（预算限制）**
        * **$0.8x + 0.2y \leq 60000$（风险限制）**
        * **$x \geq 0, y \geq 0$（非负约束）**
+ **理财应用**：
    - **在10万元预算内，投资股票和债券，使收益最大**
    - **在保证年收益率6%的前提下，最小化风险**
    - **多资产投资组合的优化**
+ **教学方法**：
    - **绘制可行域**
    - **找出最优解（通常在顶点处）**
    - **进行敏感性分析**
    - **讨论实际约束条件的影响**

#### **模块四：概率与统计（3-4课时）**
**课题1：概率论与风险评估**

+ **数学知识**：
    - **期望值：$E(X) = \sum x_i \cdot p_i$**
    - **方差：$D(X) = E(X^2) - [E(X)]^2$**
    - **标准差：$\sigma = \sqrt{D(X)}$**
    - **变异系数：$CV = \frac{\sigma}{E(X)}$**
+ **理财应用**：
    - **投资A有60%概率赚10%，40%概率亏5%**
    - **投资B有50%概率赚8%，50%概率亏3%**
    - **比较两个投资的期望收益和风险**
    - **股票日收益率的概率分布分析**
    - **投资组合的风险度量**
+ **教学方法**：
    - **计算期望收益**
    - **计算风险度量（方差、标准差）**
    - **绘制概率分布曲线**
    - **进行风险-收益权衡分析**

**课题2：统计学与市场分析**

+ **数学知识**：
    - **描述性统计：平均数、中位数、众数、方差、标准差**
    - **相关性分析：相关系数、散点图**
    - **线性回归：趋势线、预测**
    - **正态分布与置信区间**
+ **理财应用**：
    - **分析过去5年的股票收益率，预测未来收益**
    - **两只基金的风险对比分析**
    - **市场趋势分析与预测**
    - **投资组合的相关性分析**
+ **教学方法**：
    - **使用真实的金融数据**
    - **制作数据可视化图表**
    - **进行趋势分析和预测**
    - **讨论统计结论的局限性**
